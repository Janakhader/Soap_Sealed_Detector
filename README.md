# Soap Package Detector

An AI-powered computer vision project that detects whether a soap package is **opened** or **closed** using deep learning and object detection techniques. The model analyzes images or video frames to classify the packaging state, making it suitable for quality control and automated inspection in manufacturing environments.

## Features

* Detects soap packages in real time
* Classifies each package as **Opened** or **Closed**
* Supports image and video input
* Built using the YOLO object detection framework
* High-speed inference for industrial applications

## Technologies Used

* Python
* YOLO (Ultralytics)
* PyTorch
* OpenCV
